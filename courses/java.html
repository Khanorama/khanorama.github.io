Java BOOTCAMP

The Java BOOTCAMP is a week long, faster-paced course meant for students with prior experience in another programming language. Here, you will be equipped with a solid foundation in Java programming. Throughout the course, we will explore a variety of essential concepts and skills that form the foundation of programming in Java, solidifying the Object-Oriented (OOP) structure of the language. At the end of this course, all students will receive a certificate upon passing the final exam.
